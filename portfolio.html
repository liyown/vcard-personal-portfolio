  • Addressing Data Heterogeneity in Federated Learning via Feature Attention Fusion

    科研项目

    Federated learning is a distributed machine learning paradigm designed to train machine learning models by protecting the data privacy of participants. The main idea is to perform model training on the local device and aggregate the updated model parameters into a global model, thus avoiding the transmission of raw data. However, due to the heterogeneity of data distribution among participants, the local training of the global model may cause the local model to shift toward local data. This paper proposes a new federated feature attention fusion algorithm (FedFormer) to alleviate this problem by fusing global features and local features. Furthermore, we design a multi-layer feature fusion method (MC-FFAF) to obtain feature expression capabilities from multiple levels. This method uses the attention mechanism to calculate the correlation between global features and local features, and selects features with high correlation to be integrated into the local model. Experimental results show that MC-FFAF has significantly improved the original training framework in terms of convergence speed and model accuracy.

  • Class Imbalance Problem in Federated Learning: Addressing it through Inter-Class Information Mining

    科研项目

    In view of the existing methods, the relationship between samples has not been deeply discussed. Because the relationships between different samples are diverse and complex, it is necessary to consider how to use the information in the majority class to assist the learning of the minority class. Our work is devoted to exploring how to optimize federated learning models to better handle relationships between samples, especially when dealing with unbalanced class distributions. Our focus is on how to use the information in the many classes to assist the learning of the few classes to achieve more comprehensive and accurate model performance. To this end, we propose a new method, which optimizes the federated learning model by mining the relationships between samples and using the information of most classes. Our work aims to provide new ideas and methods for improving the performance of federated learning in the case of complex data distribution.

  • get_bibtexs

    简便重复工作

    本项目旨在简化在使用 LaTeX 写论文时将引用转换为 BibTeX 格式的繁琐过程。尤其是当文献量很大时，手动转换引用的工作实在不值得。虽然一些文献管理工具如 EndNote 和 Zotero 提供了方便的导出功能，但如果没有这些工具，我们提供了一个解决方案：封装各个文献数据库的API，提供统一的接口，使用户能够轻松地获取引用信息并将其转换为 BibTeX 格式。这样，无论你使用何种文献管理工具，都能够轻松地管理你的引用，并在 LaTeX 论文中方便地引用这些文献。
